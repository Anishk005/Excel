## Overview

This project demonstrates an end-to-end E-Commerce Sales Analysis workflow using Microsoft Excel. Starting with a raw, unstructured e-commerce dataset, the project covers data cleaning, transformation, analysis, visualization, and the creation of an interactive dashboard.

The objective is to simulate a real-world business scenario where raw sales data is converted into meaningful insights that support data-driven decision-making.

##Objectives
Clean and prepare messy sales data for analysis.
Perform sales analysis using Excel formulas and PivotTables.
Build an interactive dashboard with KPIs and slicers.
Generate business insights from e-commerce transactions.
Demonstrate Excel skills relevant to Data Analytics and Business Intelligence roles.
